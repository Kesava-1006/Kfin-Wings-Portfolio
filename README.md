# SIP Portfolio Management System

A modern SIP (Systematic Investment Plan) portfolio management application built with **Next.js**, designed to manage investors, create SIPs, track transactions, and visualize portfolio data in a clean dashboard interface.

---

# Features

* Secure Login Authentication
* Investor Creation & Management
* SIP Creation & Tracking
* Portfolio Dashboard
* Personal Dashboard View
* Transaction History
* Data Storage using LocalStorage
* Token Management using Cookies
* Data Fetching from Supabase
* Responsive UI with modern design

---

# Tech Stack

* **Frontend:** React.js / Next.js
* **Styling:** Tailwind CSS
* **Backend/API:** Node.js / Express
* **Database:** Supabase
* **Authentication:** Cookies + LocalStorage

# Installation & Setup

## Clone the Repository

```bash
git clone <your-repository-url>
cd <project-folder>
```

## Install Dependencies

```bash
npm install
```

## Run the Development Server

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
